# Rhyme Game

A fun, interactive rhyming game that challenges players to come up with words that rhyme with the displayed word.

## How to Play

1. The game will display a random word.
2. Type a word that rhymes with the displayed word using the virtual keyboard or your physical keyboard.
3. Press the SEND button or hit Enter to submit your answer.
4. If your word rhymes correctly (has the same final vowel sound), you'll earn a point and get a new word.
5. If your word doesn't rhyme, your streak will reset to zero.
6. Try to build the longest streak possible!

## Features

- Virtual keyboard for mobile devices
- Word definitions available by clicking the "Definition" button
- Streak counter to track your progress
- Fun military-themed feedback messages
- Animated word effects

## Installation

This game is a standalone HTML/CSS/JavaScript application. To run it:

1. Simply open the `index.html` file in a web browser.
2. No server or additional installation is required.

## Technical Details

- Built with HTML, CSS, and JavaScript
- Uses Phaser 3 for game animations and effects
- Fetches words and definitions from external APIs (datamuse.com and dictionaryapi.dev)
- Responsive design works on both desktop and mobile devices
